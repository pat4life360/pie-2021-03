VRCar

PIE Final Project '21

A remote-controlled car where the user can view around from it in real-time VR in all 360 degrees.

Team: Aaron Huang,  Jaclyn Ho, Lauren Xiong, Leopoldo Sanchez, and Melissa Kazazic.

about us

The Overview

What is the goal of the VRCar?

vr viewpoint

The user is able to view from the point of the car's view, in all 360 degrees, in real-time. This allows the user to see the envorinemnt surrounding the car when viewing and controlling the car.

more
stable viewing

Create a damping system for the camera that allows for stable viewing, and the shakes of the car don't affect the view. Include distance sensors to stop the car from bumping into walls and other objects.

more
remote-controlled

Control the car while a user is in VR, using a remote controller such as an Xbox Controller. Build from a driving system such as tank drive and iterate onward. A stretch goal is attempting a telemetry system for less wires.
