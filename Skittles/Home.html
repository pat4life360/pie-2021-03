We Make Robots Make Art

We designed a pick and place machine that picks up Skittles and places them on a board to create an image. 

An Interdisciplinary Project

This project was a collaboration between mechanical, software, and electrical and computer engineers.

Mechanical

The mechanical team created a gantry system, vacuum-powered end effector, and feeders to dispense the Skittles

more
Electrical

The electrical system provides power to all components and interfaces with the firmware to control electromechanical devices

more
Firmware

Written in Arduino, the firmware controls the stepper motors, servos, and valve state using an STM Bluepill and Arduino Uno

more
Software

Images are broken down into pixels; the color of each pixel informs the motion protocol, which is interfaced with the firmware

more
